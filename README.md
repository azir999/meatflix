1. Clone my repositories with link below

```bash
git clone https://github.com/azir999/meatflix
```
2. Install composer

```bash
composer install
```

3. Migration my database and seeder
```bash
php artisan migrate --seed
```
4. Clone env

```bash
cp .env.example .env
```

5. Get key

```bash
php artisan key:generate
```

6. Run css

```bash
npm install
npm run dev
```

7. Run website

```bash
php artisan serve
```
